# Retro TTY Tree Animation

A nostalgic terminal-style landing page featuring a procedurally generated tree animation with flying birds, designed to look like a retro CRT monitor display.

## Features

- **15-second tree growth animation** - Watch a fractal tree grow line by line in classic terminal style
- **Retro CRT effects** - Scanlines, screen flicker, and phosphor glow for authentic vintage feel
- **Animated birds** - After the tree completes growth, birds fly in and orbit around the canopy
- **Interactive** - Click anywhere or press Enter to redirect to notes.phttp.com
- **Responsive** - Adapts to any screen size
- **No dependencies** - Pure HTML, CSS, and JavaScript

## Visual Style

- **Color**: Classic terminal green (`#00ff41`) on black background
- **Font**: VT323 monospace font for authentic terminal typography
- **Effects**: Horizontal scanlines, screen curvature vignette, subtle flicker animation

## Usage

1. Save the code as `index.html`
2. Open in any modern web browser
3. Wait for the animation to complete (15 seconds)
4. Click or press Enter to continue

## Customization

| Parameter | Location | Description |
|-----------|----------|-------------|
| `treeDuration` | JavaScript | Animation duration in milliseconds (default: 15000) |
| `NUM_BIRDS` | JavaScript | Number of birds to spawn (default: 8) |
| `COLOR_MAIN` | JavaScript | Primary terminal color (default: `#00ff41`) |
| `redirect URL` | JavaScript | Change destination in `redirectToGoogle()` function |
